🍊 VitalisCare - Wellness & Health Portal

VitalisCare is a premium, high-performance medical landing page designed with a focus on Vitality, Warmth, and Energy. By moving away from traditional "cold" clinical blues, this project uses a high-converting Orange & Amber palette to represent holistic wellness, energy, and approachable healthcare.

🎨 Design Psychology (The Orange Shift)

The color scheme was intentionally shifted to Orange (#f97316) and Amber (#fbbf24) to:

Encourage Optimism: Orange is the color of health, vitamins, and morning energy.

Increase Conversions: Warm colors are psychologically proven to drive more clicks on "Book Now" and "Emergency" buttons.

Holistic Feel: Perfect for modern clinics that focus on both physical medicine and overall wellness.

🚀 Key Features

Urgent Care Integration: A high-visibility "Specialists Online" status indicator.

Appointment Engine: A streamlined, 60-second booking form with specialty selection.

Trust Indicators: Real-time patient ratings (4.9/5) and verification badges.

Smooth Scroll Reveal: Elements fade and slide into view as the patient scrolls, creating a premium experience.

Fully Responsive: Custom CSS Grid and Flexbox layouts that adapt perfectly to iPhones, iPads, and Android devices.

🛠️ Tech Stack

HTML5: Structured for SEO and screen-reader accessibility.

CSS3: Featuring custom variables for easy global color changes.

Vanilla JavaScript: Lightweight intersection observers for animations and form validation.

Google Fonts:

Outfit: For bold, authoritative headings.

Plus Jakarta Sans: For clean, easy-to-read medical copy.

📂 Customization: Changing the Orange Scheme

The website is built using CSS Root Variables. If you want to adjust the intensity of the orange, look for this block in the <style> section:

code

:root {
    --primary: #f97316;    /* Main Vitality Orange */
    --secondary: #fbbf24;  /* Warm Amber Accent */
    --accent: #ea580c;     /* Deep Burnt Orange for hover states */
    --bg: #fffcf9;         /* Off-white with a hint of warmth */
    --white: #ffffff;
    --text-dark: #1f2937;
}
